# design-harness — product one-pager

Tier 1 context — always loaded. ≤30 lines.

## What it is

A Claude Code plug-in for design teams building AI harnesses. Ships skills that explain the five-layer framework (Context / Skills / Orchestration / Rubrics / Knowledge) and scaffold one in a user's repo. Companion to a Substack article series making the thesis.

## Who it's for

**5–20 person design teams** adopting AI-assisted workflows. Secondary: solo designers and 20+ person design orgs. Not: non-design teams.

## Core value proposition

Turn scattered AI-context pile (Slack pins, Notion docs, Figma comments, retros) into a working five-layer harness so design knowledge **compounds** rather than rots across designer rotation.

## Out of scope (non-goals)

- Not a full design tool replacement (sits around Figma/Claude/etc., not instead of)
- Not a SaaS (local-first, file-based, git-versioned)
- Not a curriculum (the Substack article teaches; plug-in ships a working example)
